# Features

- Frame acquisition
- Preprocessing and lane detection
- Object detection and tracking
- Risk assessment
- Rendering of overlays
- Live webcam input
- Video file selection
- Drag-and-drop path input
- Continuous rendering loop for real-time display

  ![detectfront](https://github.com/user-attachments/assets/0057df39-7f21-41a5-b490-8339282b20b0)


# Requirements

- Ensure that a compatible model file is available in the working directory or provide a valid path.

# Usage

- Run the application:

      python main.py

- The system automatically resets tracking and lane estimation when a new source is loaded.

# Disclaimer

- Outputs such as collision warnings are probabilistic and should not be treated as definitive.
